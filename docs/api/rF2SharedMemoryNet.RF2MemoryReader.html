Remarks

The RF2MemoryReader class is designed to interface with rFactor2's shared memory plugin, allowing access to telemetry, scoring, rules, force feedback, graphics, pit information, weather, and other data. It attempts to open the necessary memory-mapped files upon instantiation and provides methods to read data synchronously and asynchronously. The class implements IDisposable to ensure proper release of resources.

RF2MemoryReader(ILogger?)

Initializes a new instance of the RF2MemoryReader class, which attempts to open memory-mapped files used by the rFactor2 game for telemetry and control data.

public RF2MemoryReader(ILogger? logger = null)

Parameters

logger ILogger

An optional logger for recording errors and informational messages. If null, no logging will occur.

Remarks

This constructor attempts to open several memory-mapped files that are expected to be available when the rFactor2 game is running with the Shared Memory Plugin enabled.
If the game is not running or the plugin is not installed, an error will be logged if a logger is provided. This error can be safely ignored if the game is not currently running.
In debug it will output to console regardles if any logger is attached.
