Remarks

This class contains file names, layout versions, and other constants that are used for accessing rFactor 2 shared memory data and control interfaces. These constants are intended for use in applications that integrate with rFactor 2 telemetry, scoring, rules, force feedback, graphics, weather, and other systems.
